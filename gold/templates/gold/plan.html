Phase 0: Requirements Specification

This project will contain 2 apps, the RESTful unit conversion Api, and the gold page that uses that api and has all the ui for the project.

Create a django app called 'unitconv'. This will provide a web api that performs conversions between units. This app will have a view function called convert, which will respond to get request from a form. This form will provide the following parameters to the url: from - a string that represents the type of units of the input, value - a non-negative float representing the number of from units, to - a string representing the type of units the user wants value converted into. The result of the request will be a JSON object. When a user makes an invalid query, return a JSON object with an error property that has a string of the error message.

The calculations are made by referring to a Django model of conversion factors. Create a data migration to initialize the database with the necessary conversion factors. The unit conversion API must support the following units: U.S. ton, gram, troy ounce, kilogram, imperial pound, ounce. The API must be capable of converting between all supported units.

Next, create another app called gold the presents a single html page allowing the user to put a weight and a unit and calculate the worth in gold. The page must have instructions on how to use the app, an input to accept weight, a select drop-down to pick the unit, a button that performs the conversion when clicked, a paragraph element that displays the current price of gold. Before it is retrieved from nasdaq, have it say "please wait". The page will automagically query Nasdaq Data Link one time upon loading the page. Use fetch() to get the price of gold in us dollars and store it in a global variable so its available when the button is clicked. The button will activate a callback function which will do the following. Read the input and validate that it is acceptable. Make a fetch() request to the unitconv app that results in the users weight conversion into troy ounces. Multiply that amount by the market price of gold. Create a new div to display the result, including a time stamp. Clicking a div removes it from the dom.
